# licochron-history

## 概要
Licochon プロジェクトの正史（Seishi）および関連する歴史データを管理するためのリポジトリです。
このリポジトリは公開（Public）であり、プロジェクトの軌跡を透明に記録・提示することを目的としています。

## 構造
- `assets/history/seishi/`: 確定した正史データ（手記、手紙など）。
- `assets/history/variants/`: 並行世界や異説の記録。
- `assets/appendix/`: 歴史を補足する資料。

## 運用
このリポジトリへの主要な記録は Lico (Iuria) および編纂者 (Hasta) によって行われます。
他者からの寄稿（プルリクエスト）は、平行世界ブランチへの統合または正史へのマージという形で受け入れられます。
